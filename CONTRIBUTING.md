# Contributing

Thanks for your interest in improving this repository.

## Scope

This repository is a structured MVP for applying Lean Six Sigma logic with AI-assisted workflows. Helpful contributions improve clarity, rigor, usability, and artifact quality.

## Good contributions

Examples include:
- improving phase logic,
- strengthening templates,
- refining evidence discipline,
- adding realistic examples,
- improving role-aware outputs,
- correcting inaccuracies.

## Contribution expectations

Please keep contributions:
- concise,
- implementation-aware,
- useful in real project settings,
- free of proprietary or confidential material.
