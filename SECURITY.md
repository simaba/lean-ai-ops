# Security Policy

Thank you for helping improve the security and integrity of this repository.

## Scope

This repository is maintained as a public portfolio and open knowledge resource.
It may contain lightweight reference code, templates, and framework material.

If you believe you have found a security issue related to code, scripts, dependency configuration, or unsafe example behavior, please report it privately.

## How to report a security issue

Please do not open a public issue for suspected security problems.

Instead, use a GitHub private security advisory, if enabled, or open a public issue only if the matter is non-sensitive and does not expose exploit details.
